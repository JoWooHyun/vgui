## 1. 안전 주의사항

### 경고

- 프린터를 습하거나 물이 닿는 곳에 설치하지 마십시오.
- 전원 코드가 손상된 경우 즉시 사용을 중지하십시오.
- 프린터 작동 중에는 내부에 손을 넣지 마십시오. 모터 및 블레이드에 의해 부상의 위험이 있습니다.
- UV LED가 켜져 있을 때 직접 노출된 빛을 눈으로 보지 마십시오.

### 주의

- 세라믹 레진을 취급할 때는 반드시 장갑을 착용하십시오.
- 레진이 피부에 묻은 경우 즉시 비누와 물로 씻어내십시오.
- 환기가 잘 되는 장소에서 사용하십시오.
- 프린터를 수평한 곳에 설치하십시오. 기울어진 상태에서는 레진이 고르게 펴지지 않습니다.
- 임의로 분해하거나 개조하지 마십시오.

## 2. 제품 소개

### 개요

MAZIC CERA는 DLP(Digital Light Processing) 방식의 3D 세라믹 프린터입니다. 세라믹 슬러리를 자동으로 토출하고 블레이드로 평탄화한 뒤, UV LED로 경화하여 3D 형상을 적층합니다.

### 주요 구성

| 번호 | 구성 | 설명 |
|------|------|------|
| 1 | 빌드 플랫폼 (Z축) | 조형물이 적층되는 스테이지. 상하 이동. |
| 2 | 블레이드 (X축) | 레진을 평탄화하는 금속 블레이드. 좌우 이동. |
| 3 | 레진 피더 (Y축) | 시린지 펌프로 레진을 토출하는 장치. |
| 4 | DLP 프로젝터 | 하단에서 UV 패턴을 투영하여 레진을 경화. |
| 5 | 터치 LCD | 7인치 터치스크린. 모든 조작은 이 화면에서 수행. |
| 6 | USB 포트 | 출력 파일(.zip)을 읽기 위한 USB 메모리 연결. |

### 작동 원리

```
레진 토출 → 블레이드 평탄화 → UV 노광(경화) → 플랫폼 상승 → 반복
```

1. 시린지에서 레진을 토출합니다.
2. 블레이드가 레진을 균일하게 펴줍니다.
3. 프로젝터가 해당 레이어의 이미지를 투영하여 레진을 경화합니다.
4. 플랫폼이 한 레이어 높이만큼 상승합니다.
5. 위 과정을 반복하여 3D 형상을 완성합니다.

## 4. 터치스크린 안내

모든 조작은 7인치 터치스크린에서 수행합니다.

### 4.1 홈 화면

전원을 켜면 홈 화면이 표시됩니다.

```
┌─────────────────────────────────────────────────┐
│                                                   │
│                  MAZIC CERA                        │
│                                                   │
│       [ Tool ]     [ System ]     [ Print ]       │
│                                                   │
│                                       VERICOM     │
└─────────────────────────────────────────────────┘
```

| 버튼 | 기능 |
|------|------|
| **Tool** | 수동 제어, 노출 테스트, 레벨링, 설정, 소재 관리 |
| **System** | 장치 정보, 언어, 테마, 서비스 연락처 |
| **Print** | USB에서 파일을 선택하여 출력 시작 |

### 4.2 숫자 입력 방법

거리, 속도 등의 값을 설정할 때는 값이 표시된 버튼을 터치합니다.

1. 버튼을 터치하면 **숫자 키패드**가 화면에 표시됩니다.
2. 숫자를 입력하고 **확인**을 누릅니다.
3. 최소/최대 범위를 벗어나면 자동으로 조정됩니다.

### 4.3 화면 이동

- 각 페이지의 좌측 상단 **← (뒤로가기)** 버튼으로 이전 화면으로 돌아갑니다.
- 홈 화면은 모든 기능의 시작점입니다.

### 4.4 화면 구조

```
홈 화면
 ├── Tool (도구)
 │    ├── Manual (수동 제어)
 │    ├── Exposure (노출 테스트)
 │    ├── Leveling (레벨링)
 │    ├── Setting (LED/블레이드/레진 설정)
 │    └── Material (소재 프리셋 관리)
 │
 ├── Print (출력)
 │    ├── 파일 선택
 │    ├── 파일 미리보기
 │    └── 출력 진행
 │
 └── System (시스템)
      ├── Device Info (장치 정보)
      ├── Language (언어)
      ├── Theme (테마)
      └── Service (A/S 연락처)
```

## 5. 출력 준비

출력하기 전에 아래 3가지를 순서대로 수행하십시오.

### 5.1 레벨링 (최초 1회)

빌드 플랫폼과 블레이드의 높이를 맞추는 작업입니다. 장비 조립 후 또는 부품 교체 후 1회 수행합니다.

**경로:** 홈 → Tool → Leveling

| 단계 | 화면 안내 | 해야 할 일 |
|------|----------|-----------|
| Step 1 | Z축 홈 이동 | 스테이지를 올려놓되 **나사를 조이지 마십시오**. 버튼을 눌러 Z축 홈잉 실행. |
| Step 2 | X축 홈 이동 | 블레이드를 장착하고 **나사를 풀어 놓으십시오**. 버튼을 눌러 X축 홈잉 실행. |
| Step 3 | X축 중앙 이동 | 블레이드가 중앙(75mm)으로 이동합니다. 버튼을 눌러 실행. |
| 완료 | 고정 안내 | **블레이드 나사를 먼저 고정** → **스테이지 나사를 고정** → **Done** 버튼 터치. |

> **중요:** 반드시 블레이드를 먼저 고정한 후 스테이지를 고정하십시오.

### 5.2 소재 설정

출력할 소재에 맞는 파라미터를 확인하고 필요 시 수정합니다.

**경로:** 홈 → Tool → Material

1. 왼쪽 프리셋 목록에서 사용할 소재를 선택합니다.
2. 오른쪽에서 각 파라미터를 확인합니다.
3. 값을 터치하면 숫자 키패드로 수정할 수 있습니다.
4. 수정 사항은 자동으로 저장됩니다.

**기본 내장 소재:**

| 소재 | 블레이드 속도 | LED 파워 | 토출 거리 | 토출 속도 | 토출 딜레이 |
|------|-------------|---------|----------|----------|------------|
| Zirconia | 5 mm/s | 43% | 1.0 mm | 3 mm/s | 5.0초 |
| Alumina | 5 mm/s | 50% | 1.0 mm | 3 mm/s | 5.0초 |
| Hydroxyapatite | 4 mm/s | 45% | 1.2 mm | 3 mm/s | 5.0초 |

**파라미터 설명:**

| 파라미터 | 범위 | 설명 |
|----------|------|------|
| Blade Speed | 1~100 mm/s | 블레이드가 레진을 펴는 속도 |
| LED Power | 9~100% | UV LED 노광 밝기 |
| Resin Dist. | 0.1~5.0 mm | 레이어당 레진 토출량 (거리) |
| Resin Speed | 1~15 mm/s | 레진 토출 속도 |
| Resin Delay | 0.5~300초 | 토출 후 레진 안정화 대기 시간 |
| Pull Dist. | 0.0~5.0 mm | 토출 후 되돌리기 거리 (0=비활성) |
| Pull Delay | 0.1~20.0초 | 되돌리기 구간 시간 |
| Return Dist. | 0.0~5.0 mm | 되돌린 후 다시 밀기 거리 (0=비활성) |
| Return Delay | 0.1~20.0초 | 다시 밀기 구간 시간 |

새로운 소재를 추가하려면 **[+ 추가]** 버튼을 누르고 이름을 입력합니다.

### 5.3 프라이밍

레진 토출구(노즐)에 레진을 미리 채워놓는 작업입니다. 출력 전에 반드시 수행하십시오.

**경로:** 홈 → Tool → Setting → RESIN 영역

1. **PRIME START** 버튼을 터치합니다. Y축이 홈으로 이동합니다.
2. **MOVE** 버튼으로 Y축을 조금씩 이동시킵니다.
3. 노즐 끝에서 레진이 나오기 시작하면 **PRIME DONE**을 터치합니다.
4. 현재 Y축 위치가 저장되어 출력 시 이 위치부터 토출이 시작됩니다.

> **주의:** 프라이밍을 하지 않으면 출력 초반에 레진이 토출되지 않아 조형이 실패할 수 있습니다.

## 6. 출력하기

### 6.1 파일 준비

슬라이서에서 모델을 슬라이싱한 후 `.zip` 파일을 USB 메모리에 저장합니다.

### 6.2 출력 시작

**경로:** 홈 → Print

**Step 1. 파일 선택**

```
┌──[ Print  ← ]────────────────────────────────────┐
│                                                    │
│   [파일1]  [파일2]  [파일3]        [ ↑ ]           │
│   [파일4]  [파일5]  [파일6]        [ ↓ ]           │
│                                    [Open]          │
│                                    [Home]          │
└────────────────────────────────────────────────────┘
```

1. USB 메모리를 프린터에 연결합니다.
2. 파일 목록이 자동으로 표시됩니다 (썸네일 포함).
3. 출력할 파일을 터치하여 선택합니다 (청록색 테두리 표시).
4. ↑/↓ 버튼으로 페이지를 넘깁니다.
5. **Open** 버튼을 터치합니다.

**Step 2. 파일 미리보기**

```
┌──[ File Preview  ← ]─────────────────────────────┐
│                                                    │
│   ┌──────────┐    레이어:    150                   │
│   │          │    예상 시간:  45분                  │
│   │ (썸네일)  │    레이어 높이: 0.05mm               │
│   │          │    LED:       43%                   │
│   └──────────┘    블레이드:   5mm/s                 │
│                    노광시간:   8.0s                  │
│                                                    │
│              [ Start ]    [ Delete ]               │
└────────────────────────────────────────────────────┘
```

1. 출력 파라미터를 확인합니다.
2. **Start** 버튼을 터치하면 출력이 시작됩니다.

**Step 3. 출력 진행**

```
┌──[ Printing ]─────────────────────────────────────┐
│                                                    │
│   ┌──────┐    ┌──────────────────┐                │
│   │썸네일 │    │ 현재 레이어 이미지  │                │
│   │      │    │                  │                │
│   │15/150│    └──────────────────┘                │
│   │잔여시간│                                       │
│   └──────┘     [ Pause ]   [ Stop ]               │
│                                                    │
│   [========================>       ] 10%           │
└────────────────────────────────────────────────────┘
```

화면에 현재 진행 상태가 표시됩니다.

### 6.3 출력 중 조작

| 버튼 | 동작 |
|------|------|
| **Pause** | 현재 레이어 완료 후 일시정지합니다. 버튼이 **Resume**으로 바뀝니다. |
| **Resume** | 일시정지를 해제하고 출력을 재개합니다. |
| **Stop** | 출력을 중지합니다. 확인 팝업이 표시됩니다. |

### 6.4 출력 완료

출력이 완료되면 "완료되었습니다!" 팝업이 표시됩니다.
필요한 경우 **Z Home** 버튼으로 플랫폼을 원점으로 이동시킨 후 조형물을 꺼냅니다.

### 6.5 레진 부족 시

출력 중 레진이 부족하면 자동으로 감지되어 화면에 선택 버튼이 표시됩니다.

```
┌──[ 레진 부족 ]───────────────────────────────────────┐
│                                                       │
│   [주사기 리필]    [수동배급]    [ STOP ]              │
│     (초록)         (시안)       (빨강)                │
│                                                       │
└───────────────────────────────────────────────────────┘
```

| 버튼 | 동작 |
|------|------|
| **주사기 리필** | 새 시린지로 교체한 후 터치합니다. 프린터가 자동으로 현재 Y축 위치를 읽어 토출을 재개합니다. |
| **수동배급** | 레진을 수동으로 공급하는 경우 선택합니다. Y축 토출이 비활성화되고 대기시간만 유지하며 출력이 계속됩니다. |
| **STOP** | 출력을 중지합니다. |

### 6.6 출력 시퀀스 상세

각 레이어마다 아래 순서가 자동으로 실행됩니다.

```
① Z축 이동       → 해당 레이어 높이로 플랫폼 이동
② 레진 토출 (3단계 Push-Pull)
   Push          → Y축 -방향으로 설정 거리만큼 레진 토출
                    (레진 부족 시 즉시 감지 → 사용자 선택)
   안정화 대기    → 토출 딜레이 시간 동안 대기
   Pull          → Y축 +방향으로 되돌리기 (설정 시, 0이면 스킵)
   Return        → Y축 -방향으로 다시 밀기 (설정 시, 0이면 스킵)
③ 블레이드 스윕   → X축 10mm → 140mm (레진 평탄화)
④ UV 노광        → LED ON, 설정 시간 동안 경화
⑤ LED OFF        → 노광 종료
⑥ Z축 리프트     → 플랫폼 +5mm 상승
⑦ 블레이드 복귀   → X축 140mm → 10mm (50mm/s 고정)
→ 다음 레이어 반복
```

### 7.1 Manual — 수동 제어

각 모터 축을 직접 제어하여 위치를 확인하거나 조정합니다.

**경로:** 홈 → Tool → Manual

```
┌──[ Manual  ← ]───────────────────────────────────┐
│                                                    │
│   Z Axis        X Axis (Blade)    Resin Feeder    │
│   [1.0 mm]      [10.0 mm]         [1.0 mm]        │
│    [ ▲ ]        [5 mm/s]           [ ▲ ]          │
│    [ ▼ ]        [ ◀ ][ ▶ ]        [ ▼ ]          │
│   [HOME]        [HOME]            [HOME]          │
│                                                    │
└────────────────────────────────────────────────────┘
```

#### Z축 (빌드 플랫폼)

빌드 플랫폼의 상하 위치를 조정합니다.

| 조작 | 설명 |
|------|------|
| 거리 버튼 터치 | 이동 거리 입력 (0.05~100mm) |
| ▲ | 플랫폼 상승 |
| ▼ | 플랫폼 하강 |
| HOME | 원점(최하단)으로 이동 |

#### X축 (블레이드)

블레이드의 좌우 위치를 조정합니다.

| 조작 | 설명 |
|------|------|
| 거리 버튼 터치 | 이동 거리 입력 (최대 140mm) |

### 7.2 Exposure — 노출 테스트

프로젝터의 투영 상태를 테스트하거나, 트레이를 클리닝할 때 사용합니다.

**경로:** 홈 → Tool → Exposure

```
┌──[ Exposure  ← ]─────────────────────────────────┐
│                                                    │
│   [Image] [Checker] [Logo] [20x20] [Clean]        │
│                                                    │
│          "체크무늬 패턴을 노출합니다"                │
│                                                    │
│                  [ 5 sec ]                         │
│                                                    │
│                  [ START ]                         │
└────────────────────────────────────────────────────┘
```

#### 패턴 종류

| 패턴 | 용도 | 최대 시간 |
|------|------|----------|
| **Image** | USB에서 이미지 파일을 선택하여 투영 (.png, .jpg, .bmp, .tiff) | 120초 |
| **Checker** | 체크무늬 패턴으로 투영 상태 확인 | 60초 |
| **Logo** | VERICOM 로고 패턴 투영 | 60초 |
| **20x20** | 20×20mm 사각형 투영 (투영 크기 검증용) | 60초 |
| **Clean** | 전체 화면 노출 (트레이 바닥 경화/클리닝) | 120초 |

#### 사용 방법

1. 패턴 버튼을 터치하여 원하는 패턴을 선택합니다.
   - **Image** 선택 시: 파일 선택 창이 열립니다. USB에서 이미지를 선택하십시오.
2. 시간 버튼(`5 sec`)을 터치하여 노출 시간을 설정합니다.
3. **START**를 터치하면 노출이 시작됩니다.
4. 설정 시간이 지나면 자동 정지됩니다. **STOP**으로 수동 정지할 수도 있습니다.

## 9. 유지보수 및 관리

### 9.1 트레이 클리닝

출력 후 트레이 바닥에 경화된 레진이 남아 있으면 다음 출력에 영향을 줄 수 있습니다.

1. 홈 → Tool → Exposure → **Clean** 패턴을 선택합니다.
2. 시간을 60~120초로 설정합니다.
3. **START**를 눌러 전체 화면을 노출합니다.
4. 경화된 필름을 트레이에서 조심스럽게 벗겨냅니다.

### 9.2 블레이드 관리

- 블레이드에 레진이 굳어 있으면 부드러운 천으로 닦아주십시오.
- 블레이드가 휘었거나 손상된 경우 교체하십시오.
- 블레이드 교체 후에는 반드시 **레벨링**을 다시 수행하십시오.

### 9.3 레진 보관

- 세라믹 레진은 직사광선을 피해 서늘한 곳에 보관하십시오.
- 장기간 사용하지 않을 경우 시린지에서 레진을 빼놓으십시오.
- 사용 전 레진을 충분히 교반하십시오.

### 9.4 시린지 (레진 피더) 관리

- 시린지 내부에 레진이 굳지 않도록 사용 후 청소하십시오.
- 노즐 막힘 시 가는 핀으로 막힌 부분을 제거하십시오.

## 10. 문제 해결 (FAQ)

### 프로젝터가 켜지지 않습니다

- "프로젝터 연결 실패" 팝업이 나타나면 프린터 전원을 껐다가 다시 켜십시오.
- USB 케이블 연결 상태를 확인하십시오.

### 출력물이 플랫폼에 붙지 않습니다

- 레벨링이 정확한지 확인하십시오 (Tool → Leveling).
- 바텀 레이어의 노광 시간을 늘려보십시오.
- 플랫폼 표면이 깨끗한지 확인하십시오.

### 레진이 토출되지 않습니다

- 프라이밍이 완료되었는지 확인하십시오 (Tool → Setting → PRIME DONE).
- 시린지에 레진이 충분한지 확인하십시오.
- 노즐이 막혀 있지 않은지 확인하십시오.

### 블레이드가 레진을 고르게 펴지 못합니다

- 블레이드 속도를 낮춰보십시오 (Tool → Material에서 Blade Speed 조정).
- 레진 토출량(Resin Dist.)을 조정해보십시오.
- 레진 딜레이 시간을 늘려 레진이 충분히 퍼지도록 하십시오.
- 블레이드가 휘었거나 손상되지 않았는지 확인하십시오.

### USB 파일이 표시되지 않습니다

- USB 메모리가 올바르게 연결되었는지 확인하십시오.
- 파일 형식이 `.zip`인지 확인하십시오.
- USB를 뽑았다가 다시 연결해보십시오.

### 출력 중 레진 부족 알림이 표시됩니다

- **주사기 리필**: 새 시린지로 교체한 후 "주사기 리필" 버튼을 터치합니다.
- **수동배급**: 레진을 직접 수동으로 공급하는 경우 "수동배급" 버튼을 터치합니다.
- 레진 보충이 어려우면 **STOP** 버튼으로 출력을 중지합니다.

### 출력물 표면에 줄무늬가 나타납니다

- 블레이드 속도를 낮춰보십시오.
- 레진 딜레이 시간을 늘려보십시오.
- 트레이 바닥면이 깨끗한지 확인하십시오.
